关于本系统 / About DysPOLNet

模型说明、训练数据、性能指标与使用范围

Model Card
模型名称
DysPOLNet（HKU 牙医学院 Oral Cancer Research Group 开源）
原始任务
二分类：口腔白斑（Oral Leukoplakia）临床照片 → 异型增生（OED）存在 / 不存在。金标准为活检组织病理。
骨干网络
EfficientNet-B2（ImageNet 预训练 → 临床数据微调）
+ GlobalAveragePooling2D + BatchNorm + Dropout(0.3) + Dense(1, sigmoid)
概率校准
Platt Scaling（sklearn LogisticRegression），用于把 sigmoid 原始输出映射到更接近真实概率的数值
解释性输出
Grad-CAM 热力图（目标层：global_average_pooling2d）
输入要求
JPG / PNG，300×300 输入分辨率（系统自动缩放）；建议为口腔病灶近距离临床照，避免开口器、牙齿、口镜遮挡
训练样本
2,073 张回顾性口腔白斑临床照片
来源：HKU 牙科门诊 + 印度 King George's Medical University + 尼日利亚合作机构
许可
MIT License（可商用、可修改、需保留原作者署名）

风险等级映射规则

本前端根据 Platt 校准概率把模型的二分类输出映射成 3 级风险展示

低风险
校准概率 < 10%
论文中"高灵敏度（≥95%）"工作点 ↑，建议常规随访
中风险
10% ≤ 校准概率 < 50%
建议结合临床进一步检查 / 缩短复诊间隔
高风险
校准概率 ≥ 50%
原模型二分类正阳性区，建议活检明确病理

适用范围与局限

✓ 适用于：口腔白斑（已临床诊断）患者的异型增生概率辅助评估，用于决定是否需要活检 / 缩短复诊间隔 / 患者自我监测的研究与教学演示。

✗ 不适用于：(1) 替代活检与病理科医生诊断；(2) 直接判定恶性肿瘤；(3) 异型增生组织学级别（mild / moderate / severe）的细分；(4) 非白斑性口腔病变；(5) 训练数据未充分覆盖的人群与拍摄条件（如非临床近距离照、儿童病例）。

⚠ 重要声明：本系统为 HKU 开源 DysPOLNet 模型的非临床演示部署，不构成医疗诊断意见，不能替代口腔病理专科医生的临床判断。所有临床决策须由具资质的医务工作者作出。
